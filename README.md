# 🏛️ CivicAI – AI-Powered Public Grievance & Governance Assistant

## 📌 Overview
CivicAI is an AI-powered governance assistant that helps citizens report, classify, prioritize, and route civic complaints intelligently. It leverages IBM watsonx Orchestrate and IBM Granite models to automate grievance management and improve public service delivery.

## 🚀 Features

- 📝 Complaint reporting through Text
- 🎤 Voice-based complaint submission (Speech-to-Text)
- 🖼️ Image-supported complaint reporting
- 🤖 AI-powered complaint classification
- ⚡ Dynamic priority assignment (Low, Medium, High, Emergency)
- 🏢 Automatic department routing
- 📄 Structured complaint summaries
- 🔍 Duplicate complaint detection
- 📍 Civic hotspot identification
- 🔊 Text-to-Speech responses
- 📱 Responsive and user-friendly interface

## 🎯 How It Works
1. Citizen submits a complaint.
2. AI understands the complaint.
3. Complaint is classified.
4. Priority is assigned.
5. Correct government department is identified.
6. Structured complaint summary is generated.
7. Citizen receives transparent guidance.
